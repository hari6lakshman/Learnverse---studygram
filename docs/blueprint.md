# **App Name**: LearnVerse - Studygram

## Core Features:

- Student Resources: Categorize and display educational YouTube videos and descriptions by class, subject, and topic for school and college students.
- Competitive Exam Prep: Curate and organize YouTube videos and descriptions by exam, subject, and topic for competitive exam preparation.
- Motivation & Mentorship: Provide a section dedicated to motivational and mentorship videos tailored for competitive exam aspirants.

## Style Guidelines:

- Background color: Deep, rich blue (#336699) for a sophisticated learning environment.
- Card Background: Slightly darker blue (#336680) for the main content container.
- Primary/Accent Color: Vibrant gold (#D1A033) for borders, headers, and interactive elements.
- Body Text/Foreground: Soft, off-white (#D6DADF) for readability against the dark background.
- Headline font: 'Cinzel', a classic serif font, for main titles.
- Body text font: 'Lato', a clean sans-serif font, for body text and descriptions.
- Central card-like container with large rounded corners and a gold border.
- Stylish horizontal gradient separators (transparent to gold) for header and footer.
